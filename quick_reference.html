üåä Swarm Defense Checklist

Pre-Swarm Setup (30 seconds before)

  • ‚òê Find defensive position with good sightlines
  • ‚òê Place ALL defense kits (auto-turrets, mortars, grids)
  • ‚òê Drop Fixer ammo bags at position
  • ‚òê Set Hellraiser C4 at choke points
  • ‚òê Place Exterminator fire at entrances
  • ‚òê Reload all weapons
  • ‚òê Assign lanes/sectors to teammates
  • ‚òê Check for nearby medkits

During Swarm

Priority Targets:
  1. Screamers (always first!)
  2. Pyramid bases
  3. Special zombies
  4. Climbers
  5. Regular horde
Emergency Actions:
  • Masking grenade if overrun
  • Explosive at pyramid base
  • Call for help early
  • Fall back if breached
  • Save heavy weapons for crisis

üì¢ Essential Callouts & Communication

Special Zombie Callouts

Immediate Priority:
  • "SCREAMER LEFT!" - Direction critical
  • "Bull charging!" - Get ready to dodge
  • "Gasbag close!" - Headshots only
Awareness Calls:
  • "Lurker behind" - Check six
  • "Infector on roof" - Watch for spit
  • "Bomber in crowd" - Don't body shot

Tactical Communication

Situation Callout Team Response
Pyramid forming "Pyramid left wall!" Focus fire base / Use explosive
Out of ammo "Need ammo!" Fixer drops bag / Cover player
Low health "Need healing!" Medic prioritizes / Find medkit
Pinned "Help!" / "Pinned!" Immediate rescue
Breach "They're through!" Fall back / Reinforce

‚ùå Top 20 Mistakes to Avoid

Combat Mistakes

  1. Body shooting Bombers
  2. Using fire on Gasbags
  3. Ignoring Screamer audio
  4. Standing still while shooting
  5. Not checking behind (Lurkers)
  6. Wasting heavy weapons
  7. Friendly fire with explosives
  8. Melee against groups
  9. Reloading during swarms
  10. Poor pyramid targeting

Tactical Mistakes

  1. No defensive setup
  2. Hoarding equipment
  3. Solo play mentality
  4. Bad positioning
  5. No communication
  6. Ignoring objectives
  7. Wrong difficulty choice
  8. Not using class ability
  9. Forgetting defense kits
  10. Running ahead alone

Quick Fixes

If This Happens... Do This!
Team is wiping repeatedly Lower difficulty, add Medic
Running out of ammo Add Fixer, use melee more
Overrun by swarms Better positioning, use equipment
