üí° Tips for Using Mutators

  • Reward Penalties: Be aware that starred (*) mutators reduce rewards by 50% in private lobbies
  • Stacking: Multiple penalty mutators don't stack the penalty - it's always 50% max
  • Weapon XP: You still earn weapon XP normally even with penalty mutators
  • Unique Groups: Pay attention to which mutators are in the same "unique" group - only one from each group can be active
  • Challenge Mode: Some mutators are only available when the game randomly selects them for Challenge modes
  • Horde Exclusive: Certain mutators only work in Horde mode and affect shop mechanics
  • Common Mutators: You can stack multiple "common" mutators for extreme difficulty or unique combinations
  • Experimentation: Mix and match mutators for unique gameplay experiences - some combinations create interesting synergies
